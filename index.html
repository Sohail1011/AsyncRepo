Se transfiere mediante JSON (JavaScript Object Notation) del servidor al cliente, y sucesivamente.

Qué son los verbos HTTP y cuales usamos

Los verbos HTTP son los siguientes:

  1. GET -> Para obtener los datos
  2. DELETE -> Para eliminar los datos
  3. PUT -> Para actualizar los datos
  4. POST -> Para crear los datos
