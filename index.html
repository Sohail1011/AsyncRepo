Asincronia en Javascript

Como se transfiere la información basado en el protocolo HTTP

Se transfiere mediante JSON (JavaScript Object Notation) del servidor al cliente, y sucesivamente.

Qué son los verbos HTTP y cuales usamos

Los verbos HTTP son los siguientes:

  1. GET -> Para obtener los datos
  2. DELETE -> Para eliminar los datos
  3. PUT -> Para actualizar los datos
  4. POST -> Para crear los datos

Qué es un CRUD

Create Read Update Delete, o mas conocido como CRUD, sirve para dar administración a las informaciones que tiene una base de datos.

Qué es Asincronía y Sincronía

La Sincronía sirve que el dato salga de manera estructural, mientras que la asincronia sirve en dar un tiempo a la respuesta.

Como maneja la Asincronía JS

En poner un tiempo intervalo la función.
